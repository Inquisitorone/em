# bydbot

Простой Telegram-бот с использованием aiogram.

## Установка и запуск

1. Установите Python 3.10
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Экспортируйте токен:
   ```
   export TELEGRAM_API_TOKEN=your_token_here
   ```
4. Запустите бота:
   ```
   python main.py
   ```
